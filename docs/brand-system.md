# LifeLegacy Studio Suite Brand System

## Brand Scope

This brand system applies only to LifeLegacy Studio Suite.

LifeLegacy is a genealogy workflow assistant, not a replacement genealogy application.

## Core Brand Values

- Source-first
- Human-reviewed
- Preservation-centered
- Calm and professional
- Interoperable
- Original to James Thomas

## Color Palette

| Token | Hex | Purpose |
|---|---|---|
| Legacy Navy | #0D1B2A | Primary navigation, headers, trust anchor |
| Heritage Blue | #1B3A57 | Secondary panels and depth |
| Parchment | #F4F1E6 | Warm reading background |
| Heritage Gold | #C79A4A | Premium accent and highlights |
| Sage Leaf | #5F7F6B | Success, connection, continuity |
| Archive Charcoal | #2B2B2B | Primary text |

## Originality Rule

The final logo must be created as an original SVG mark.

Generated concept art may be used only as directional inspiration, not as final production artwork.

## Product Statement

LifeLegacy Studio Suite assists genealogy researchers by importing records, helping extract possible facts, preserving sources, supporting human review, and exporting to the genealogy applications researchers already use.
